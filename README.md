# 🚗 Carola | AI Destekli Araç Kiralama Sistemi

Carola, araç kiralama sürecini daha hızlı, akıllı ve kullanıcı dostu hale getirmek amacıyla geliştirilmiş bir web uygulamasıdır.  
Bu projede hedef, klasik kiralama akışlarını otomasyon ve yapay zeka ile optimize etmektir.

---

## 🎯 Proje Amacı

Kullanıcıların araç kiralama sürecinde karşılaştığı:
- Uzun ve tekrar eden form girişleri
- Karmaşık rezervasyon adımları
- Bilgi eksikliği ve kararsızlık

gibi problemleri azaltarak daha akıcı bir deneyim sunmak.

---

## 🤖 Yapay Zeka Entegrasyonu

### 📸 OCR ile Otomatik Form Doldurma
Kullanıcılar ehliyet fotoğrafını yüklediğinde, sistem gerekli bilgileri otomatik olarak okuyarak form alanlarını doldurur.

### 💬 AI Chat Desteği
OpenAI API entegrasyonu sayesinde kullanıcılar:
- Araçlar hakkında bilgi alabilir
- Kiralama sürecini öğrenebilir
- Anlık sorularına cevap bulabilir

---

## ⚙️ Sistem Özellikleri

### 🔍 Gelişmiş Filtreleme
- Yakıt tipi
- Vites türü
- Fiyat aralığı  

kriterlerine göre dinamik ve hızlı araç listeleme.

### 🧭 Çok Adımlı Rezervasyon Akışı
- Adım adım yönlendirme
- Anlık fiyat hesaplama
- Araç özeti ile şeffaf süreç

### 📊 Admin Paneli
- Araç yönetimi
- Rezervasyon yönetimi
- Kullanıcı yönetimi
- Kategori & lokasyon yönetimi

### 📧 Mail & Kupon Sistemi
Rezervasyon sonrası kullanıcıya otomatik olarak kupon kodu içeren bilgilendirme maili gönderilir.

---

## 🏗️ Teknik Mimari

### 🔹 N-Tier Architecture
- Entity Layer
- Data Access Layer
- Business Layer
- DTO Layer
- WebUI Layer  

Katmanlı yapı sayesinde sürdürülebilir, test edilebilir ve genişletilebilir bir sistem oluşturulmuştur.

### 🔹 Kullanılan Teknolojiler
- ASP.NET Core MVC
- AutoMapper
- FluentValidation
- OpenAI API (AI entegrasyonu)
- HTML / CSS / Bootstrap

---

## 🧠 Kod Kalitesi

- İş kuralları **FluentValidation** ile merkezi hale getirilmiştir  
- Katmanlar arası veri transferi **AutoMapper** ile sadeleştirilmiştir  
- Temiz, okunabilir ve sürdürülebilir bir yapı hedeflenmiştir  

---

## 🚀 Genel Bakış

Carola, sadece bir araç kiralama sistemi değil,  
kullanıcı deneyimini hızlandırmaya odaklanan bir yazılım çözümüdür.
